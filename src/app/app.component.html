Max Grundell A demo application

This is a demo applicaiton built with Angular & Firebase - API from apixu.

• The application fetches data from external API and stores it in DB under "Weather".
• User associates this data with his acc and subscribe to the "Weather" from DB.
• If data is older than 1h it will be refetched from API on DB read.

The application takes advantage of ngrx-store, to change the UI based on what time of day it is and controll dataflow.

Inspiration for this app is taken from: https://github.com/hamedbaatour/Minimus
